# Backend

## npm script plan

- `dev`: run the backend in development mode through `nodemon src/server.js` so code changes restart the server automatically.
- `start`: launch the production server via `node src/server.js`.
- `lint`: execute `eslint src --max-warnings=0` to enforce linting without warnings.
- `format`: run `prettier --write` across JS/MD/JSON files so formatting stays consistent.
